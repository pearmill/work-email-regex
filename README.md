# Work Email Regular Expressions

At [Pearmill](https://pearmill.com), we help a lot of SAAS companies acquire leads from different marketing channels.

One of the most common issues they all face is that they'd like to limit their sign up forms to only work emails.

The following are all the different regular expressions we've come to use over the past couple of years to achieve this, with different degrees of restrictions.

*Note: All of the regular expressions below are Javascript compatible and have not been tested outside of that language. You should only use them for client-side validation.*


## Ignore very popular free email sites, and common malicious form fillers

```javascript
/^([\w-\.\+]+@(?!gmail.com)(?!yahoo.com)(?!hotmail.com)(?!yahoo.co.in)(?!aol.com)(?!abc.com)(?!xyz.com)(?!pqr.com)(?!rediffmail.com)(?!live.com)(?!outlook.com)(?!me.com)(?!msn.com)(?!ymail.com)([\w-]+\.)+[\w-]{2,4})?$/i
```

## Ignore more popular free email sites, and some common typos

```javascript
/^([\w-\.\+]+@)(?!((me|mac|icloud|gmail|googlemail|hotmail|live|msn|outlook|yahoo|ymail|aol|gmail|gmail|163|wp|sky|btinternet|hatmail|email|hotmail|gnail|yahoo|protonmail|gmx|seznam|hotmail|inbox|freemail|qq|citromail|abc|xyz|pqr|rediffmail|ymail|gamil|yopmail|yandex)\.com)|((mail|yandex)\.ru)|((mail|t-online|freenet|online|hotmail|gimel)\.de)|(abv.bg|inbox.lv|cintronmail.hu|sexi.sk|email.dk|azet.sk|mail.dk|ukr.net|mail.com|gimel|netmail.eu|gmail.co))/i
```
